## ðŸ§ª Quick Start for New Users

### Install
```bash
npm install -g omniscript-cli@1.2.0
```

### Try Tables
```bash
cat > sales.osf << 'EOF'
@table {
  caption: "Q4 Sales";
  
  | Region | Revenue |
  | --- | --- |
  | North | $120K |
  | South | $95K |
}
EOF

osf render sales.osf --format html > sales.html
```

### Try Includes
```bash
# Create section
echo '@doc { # Section 1 }' > section1.osf

# Include it
cat > main.osf << 'EOF'
@meta { title: "Main Doc"; }
@include { path: "./section1.osf"; }
EOF

osf parse main.osf
```
